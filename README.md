# TalkyTeams

TalkyTeams is a tool designed to help users become more engaged and talkative during Microsoft Teams meetings.

## About

TalkyTeams aims to enhance participation and interaction during Microsoft Teams calls by providing features that encourage users to speak up and actively contribute to meetings. Whether you're dealing with meeting fatigue or just looking to be more vocal in team discussions, TalkyTeams is here to help!
